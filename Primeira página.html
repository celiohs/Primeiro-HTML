O que é HTML ?

(Voltar ao índice)

HTML ou HyperText Markup Language, é uma forma de edição de texto para exibição em navegadores, onde sé usado um padrão de normas para se chegar a um resultado, como exemplo de normas temos a utilização de tags para toda a formatação desse texto vindo a transformá-lo em código.

Principais TAGS

(Voltar ao índice)

Para realizarmos a correta formatação de um texto em HTML, utiliza-se TAGS, cada uma com sua funcionalidade específica. dentre as mais comumns temos:

  • HTML
  • HEAD
  • BODY
    • Essas são responsáveis para dar a primeira forma ao site que for desenvolver
Em sequencia podemos colocar as tags responsáveis por textualização, são elas que irão dar forma ao texto que estão inseridas na página, como exemplo :
  • H1,H2,H3,H4,H5,H6
    • Tags responsáveis por criar títilos e sub-títulos
  • OL,UL,LI
    • Tags responsáveis por criação de listas, podendo elas serem númericas ou não.
